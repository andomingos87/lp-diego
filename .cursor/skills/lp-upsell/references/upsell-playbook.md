# Playbook de Upsell — Catálogo de Oportunidades por Categoria

Este é o catálogo completo de oportunidades de over-delivery organizadas por categoria. Use como referência para identificar o que faz sentido para cada projeto específico — nem tudo se aplica a todo nicho.

---

## 1. Otimização de Conversão

Melhorias que atacam diretamente a taxa de conversão da página existente.

### Quick Wins (impacto rápido)
- **A/B Testing de headlines**: criar 2-3 variações e testar qual converte mais
- **Redesign de CTA**: cor, texto, tamanho, posicionamento baseado em boas práticas
- **Adicionar urgência/escassez**: countdown timers, vagas limitadas, oferta temporária
- **Exit intent popup**: capturar visitantes que estão saindo com oferta alternativa
- **Floating CTA**: botão fixo no mobile que acompanha o scroll
- **Social proof dinâmico**: notificações de compras recentes, número de pessoas online
- **Redução de campos no formulário**: cada campo removido pode aumentar conversão 5-10%
- **Texto de apoio no CTA**: microcopys que reduzem ansiedade ("Sem cartão de crédito")

### Melhorias Estruturais
- **Reescrita de copy com framework de persuasão** (PAS, AIDA)
- **Reestruturação de seções** para seguir sequência lógica de persuasão
- **Adição de seção de FAQ** para tratar objeções antes do CTA
- **Seção de comparação** (com/sem produto) para criar contraste
- **Calculator/estimator interativo** para nichos onde faz sentido

## 2. Ecossistema Digital

Páginas e ativos complementares que multiplicam o resultado da landing page.

### Páginas Adicionais
- **Página de obrigado otimizada**: não desperdiçar o momento pós-conversão — oferecer próximo passo, compartilhamento, upsell
- **Página de vendas longa**: para quem precisa de mais argumentos antes de converter
- **Squeeze page**: página focada 100% em captura de email/WhatsApp
- **Landing pages por persona**: variações da LP para públicos diferentes
- **Landing pages por canal**: versões otimizadas para Google Ads, Meta Ads, orgânico

### Ativos Complementares
- **Lead magnet**: ebook, checklist, planilha, mini-curso que captura leads
- **One-page site**: versão mini do site institucional para quem quer "saber mais"
- **Link in bio otimizado**: para quem vem do Instagram
- **WhatsApp Business setup**: mensagens automáticas, catálogo, status

---

## 3. Automação & Funil

Sequências e fluxos que transformam a landing page de um evento único em uma máquina contínua.

### Email Marketing
- **Sequência de boas-vindas** (5-7 emails): nutrição pós-captura
- **Sequência de carrinho abandonado**: para e-commerce
- **Sequência de reengajamento**: para leads frios
- **Newsletter template**: design de emails recorrentes
- **Setup de ferramenta**: configurar Mailchimp, RD Station, ActiveCampaign

### WhatsApp
- **Fluxo de atendimento automatizado**: chatbot básico no WhatsApp
- **Sequência de follow-up**: mensagens programadas pós-captura
- **Templates de mensagens**: para diferentes estágios do funil

### Integrações
- **Webhook landing page → CRM**: leads caem direto no sistema
- **Integração com Google Sheets**: backup automático de leads
- **Pixel e tracking**: Meta Pixel, Google Tag Manager, conversões customizadas
- **Zapier/Make automações**: conectar ferramentas do cliente

---

## 4. Conteúdo & Estratégia

Trabalho intelectual e estratégico que multiplica o valor da entrega.

### Copy Profissional
- **Reescrita completa da copy**: com pesquisa de público e framework de persuasão
- **Copy para ads**: anúncios de Meta Ads e Google Ads que direcionam para a LP
- **Script de vídeo**: roteiro para vídeo de vendas na LP
- **Copy para email**: sequências de email alinhadas com a LP

### Pesquisa & Estratégia
- **Pesquisa de concorrentes**: análise de 3-5 LPs concorrentes com comparativo
- **Mapa de persona**: documentação detalhada do público-alvo
- **Proposta de posicionamento**: como se diferenciar no mercado
- **Mapa de objeções**: todas as objeções do público + respostas para cada uma
- **Estudo de palavras-chave**: para alinhar copy com intenção de busca

### Conteúdo de Suporte
- **Blog posts otimizados para SEO**: atrair tráfego orgânico
- **Social media templates**: posts para alimentar redes sociais
- **Depoimentos estruturados**: ajudar o cliente a coletar depoimentos profissionais

---

## 5. Performance & Técnico

Otimizações técnicas que impactam conversão, SEO e experiência.

### Performance
- **Otimização de velocidade**: compressão, lazy loading, CDN, cache
- **Core Web Vitals**: LCP, FID, CLS dentro das métricas do Google
- **Otimização de imagens**: WebP, srcset, responsive images

### Tracking & Analytics
- **Google Analytics 4 setup**: com eventos customizados por seção
- **Google Tag Manager**: container com todos os pixels e eventos
- **Heatmap setup**: Hotjar/Microsoft Clarity para entender comportamento
- **Dashboard de métricas**: Google Looker Studio com KPIs da LP
- **Conversion tracking**: acompanhamento completo de conversões por fonte

### SEO
- **SEO on-page**: meta tags, schema markup, Open Graph
- **SEO técnico**: sitemap, robots.txt, canonical URLs
- **Rich snippets**: dados estruturados para resultados de busca

---

## 6. Design Premium

Elevação visual que reposiciona a marca e justifica ticket mais alto.

### Design de Marca
- **Micro-identidade visual**: paleta, tipografia e estilo visual exclusivos para a LP
- **Ícones customizados**: set de ícones desenhados para o projeto (não genéricos)
- **Ilustrações exclusivas**: ilustrações ou grafismos originais
- **Animações e micro-interações**: hover effects, scroll animations, transições suaves

### Elementos Premium
- **Vídeo background**: hero section com vídeo profissional
- **Parallax effects**: profundidade visual no scroll
- **Dark mode**: versão alternativa para experiência premium
- **Cursor customizado**: interação personalizada (para nichos criativos)
- **Loading animation**: tela de carregamento com a marca

### Responsividade Avançada
- **Design mobile-first dedicado**: layout completamente repensado para mobile
- **Adaptação para tablet**: layout específico para telas intermediárias
- **PWA**: instalar como app no celular

## Bônus de Over-delivery Gratuitos

Entregas de baixo esforço que surpreendem o cliente e geram indicações.

### Documentação
- **Guia de edição**: PDF ensinando o cliente a alterar textos e imagens sozinho
- **Checklist de lançamento**: lista do que verificar antes de colocar no ar
- **Guia de tráfego**: mini-guia de como gerar visitantes para a LP (básico)
- **Glossário de métricas**: explicação simples das métricas que o cliente vai ver

### Treinamento
- **Vídeo tutorial de 15min**: mostrando como usar/editar a plataforma
- **Mini-consultoria de tráfego**: 30min explicando como rodar anúncios básicos
- **Review mensal gratuita** (1x): um mês depois do lançamento, fazer uma análise rápida

### Templates e Recursos
- **Templates de social media**: 5-10 posts prontos no Canva para o cliente usar
- **Swipe file de emails**: 3 emails prontos para o cliente usar
- **Checklist de conteúdo**: roteiro de posts para 30 dias
- **Banco de imagens curado**: seleção de fotos/vídeos premium para o nicho do cliente
